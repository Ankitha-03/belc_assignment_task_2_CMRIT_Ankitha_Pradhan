# belc_assignment_task_2_CMRIT_Ankitha_Pradhan

## What it does

This program generates the paper folding crease pattern and returns whether the required crease is a Valley or Mountain fold.

## Approach

The solution uses recursion. For each fold, the new pattern is created using the previous pattern, adding a Valley crease in the middle, and then appending the mirrored version of the previous pattern.

## How to run

Run the program:


python task2.py


Run the test cases:


python task2_test.py
